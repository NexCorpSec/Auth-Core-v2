# Auth-Core-v2

Auth-Core-v2 is an internal authentication module developed for the NexCorp security infrastructure.

## Overview

This repository contains the core authentication logic used by internal services.
The module is designed to handle user validation, session initialization, and integration with other security components.

## Features

* Lightweight authentication module
* Designed for internal services
* Modular and easy to integrate
* Debugging and development support
